# react-native-clear-cache

This library is clear app cache for react native.

## Installation

```sh
npm install react-native-clear-cache
```

## Usage

```js
import ClearCache from "react-native-clear-cache";

// ...

const result = await ClearCache.multiply(3, 7);
```
